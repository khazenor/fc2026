# Global Game Rules:
- This modpack has the following default gamerules set each time the world is started:
  - doInsomnia: false
  - doPatrolSpawning: false
  - disableRaids: true
  - playersSleepingPercentage: 0
  - doTraderSpawning: false
  - keepInventory: true
- If you wish to disable any of these rules, you can delete the respective lines in the file `kubejs\server_scripts\global_game_rules.js`
